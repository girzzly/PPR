◆ heap_extract_min()

bool heap_extract_min ( void **  min_element)

Diese Funktion entfernt das kleinste Element aus dem Heap (dies ist immer die Wurzel des Heaps). Diese Funktion sorgt ebenfalls dafür, dass die Heap-Eigentschaft aufrechterhalten bleibt.

Parameter
min_elementZeiger für das kleinste Element im Heap.
Rückgabe
Liefet true, wenn ein Element erfolgreich extrahiert werden konnte, und false, wenn der Heap beim Aufruf bereits leer ist.
